<h1 align = "center">Aspa Tasks</h1>

### Setup to run Aspa Tasks project:

<h3> First Terminal: </h3>
- cd server
- npm i 
- npm start

<h3> Second Terminal: </h3>
- cd client
- npm i 
- npm start
